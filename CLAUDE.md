# Grayson's Lugia Game

Side-scrolling shooter for Grayson (Colin's 5yo nephew). Mobile web game hosted on GitHub Pages.

## Game Design
- Player rides Lugia, drags up/down to move
- Lugia auto-shoots projectiles
- Enemy planes fly in from the right
- 3 lives, adaptive difficulty (easier each death)
- Score counter for enemies destroyed
- Screen shake, explosions, parallax background, enemy variety
- Celebratory game over screen

## Tech
- Single HTML file with inline CSS/JS using HTML5 Canvas
- Touch-enabled for mobile phones
- Hosted on GitHub Pages (colin1201 account)
- Player sprite: grayson-lugia.png
